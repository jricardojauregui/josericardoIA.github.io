1. Exploración Inicial del Dataset

El conjunto de datos proviene del UCI Student Performance Dataset y contiene 395 filas y 10 variables con información demográfica (Escuela, Sexo, Edad), hábitos de estudio (HorasDeEstudio, Internet), historial académico (Reprobadas, Faltas) y calificaciones de tres períodos (G1, G2, G3). La variable objetivo es G3 (calificación final, escala 0–20). No se detectaron valores nulos ni outliers extremos que requirieran eliminación.

3. Análisis de Correlaciones

Observación: Existe alta correlación entre G1, G2 y G3, lo que indica multicolinealidad potencial si se incluyen las tres en el modelo. Reprobadas muestra correlación negativa moderada con G3 (≈ −0.36). El resto de variables tiene correlaciones débiles, lo que anticipa su baja significancia estadística.

4. Selección de Variables (OLS — p-valor < 0.05)

Se ajustó un modelo OLS completo con las 9 variables disponibles y se evaluó la significancia estadística de cada una. Se descartaron las variables con p-valor > 0.05:

7. Conclusiones

G2 es el predictor dominante del desempeño final (coef. ≈ 1.07), lo que confirma que el rendimiento previo es el mejor indicador del rendimiento futuro. El modelo logra R² = 0.83/0.80 (train/test), demostrando buena generalización con solo 4 variables.

La preparación de datos mediante Label Encoding y la selección estadística de variables con p-valor son pasos fundamentales para construir modelos interpretables y robustos. Como trabajo futuro, algoritmos como Random Forest o kNN podrían capturar relaciones no lineales para mejorar el rendimiento.
